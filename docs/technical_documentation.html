Executive Summary

System Overview

The Online Bookstore Management System is a modern, cloud-native microservices application built using Spring Boot 3.2.3 and Kotlin. The system provides comprehensive book catalog management, user authentication, shopping cart functionality, and order processing capabilities.

Key Features

Secure Authentication

JWT-based authentication with role-based access control supporting USER, ADMIN, and SUPERADMIN roles.

Book Management

Complete catalog with search, filtering, and real-time inventory tracking capabilities.

Shopping Experience

Full-featured shopping cart and comprehensive order processing workflow.

User Management

User registration, profile management, and administrative controls.

Real-time Updates

Event-driven architecture with Apache Kafka messaging for seamless communication.

RESTful APIs

OpenAPI 3.0 compliant REST endpoints with comprehensive documentation.

System Architecture

High-Level Architecture

The system follows a microservices architecture pattern with these key characteristics:

  • Service Independence: Each service can be developed, deployed, and scaled independently
  • Event-Driven Communication: Services communicate through Kafka events for loose coupling
  • Database per Service: Each service owns its data and database schema
  • Containerized Deployment: All services run in Docker containers

Microservices Overview

User Management Service (Port: 8083)

Purpose: Handles user authentication, authorization, and profile management.

Key Responsibilities:

  • User registration and login
  • JWT token generation and validation
  • Role-based access control (USER, ADMIN, SUPERADMIN)
  • User profile management

Database: bookstore_users

Book Inventory Service (Port: 8081)

Purpose: Manages book catalog, inventory, and search functionality.

Key Responsibilities:

  • Book catalog management (CRUD operations)
  • Inventory tracking and stock management
  • Advanced book search and filtering
  • Price management and low stock alerts

Database: bookstore_inventory

Order Management Service (Port: 8082)

Purpose: Handles shopping cart operations and order processing.

Key Responsibilities:

  • Shopping cart management
  • Order creation and processing
  • Payment confirmation
  • Order status tracking and history

Database: bookstore_orders
